# dotfiles

## tmux
- Default shell to `/bin/zsh`
- Current command is shown on the bottom of each pane
- Mouse control

## zsh
- Alias `hist` for `history`
  - Set timezone to Asia/Seoul
  - tmux window/pane doesn't share history
- Blank line after every command
